_math
=====
A sandbox for math related experiments of mine in Ruby.

```ruby
# Summation using "capital-sigma notation"
∑(1, 2, 3) # => 6.0
∑[1, 2, 3] # => 6.0

# Math::PI defaults to 15 decimals of precision
π == Math::PI # => true
π(12) == Math::PI # => false

# Use "√(num)" to get the square roots of numbers
√(4) # => 2.0
√(144) # => 12.0
```
